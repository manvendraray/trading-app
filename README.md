# Link https://forecasting-time-series.streamlit.app


This project is a web-based stock analysis and forecasting app built using Python and Streamlit.
It helps users explore historical stock prices, view technical indicators, and generate short-term forecasts using classical time-series methods.

The goal of this project is learning and analysis, not trading advice.

What does this app do?

The app allows you to:

View historical stock price data

Analyze trends using common technical indicators

Forecast the next 30 days of closing prices

Interact with clean, visual charts

Understand model performance using simple metrics

All data is fetched in real time from Yahoo Finance.
